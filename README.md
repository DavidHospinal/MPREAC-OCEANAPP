# MPREAC-OCEANAPP
## Sistema para el reconocimiento de las especies en la pesca acompañante de cerco, utilizando redes neuronales convolucionales para una empresa del sector pesquero en los puertos de Callao y Paracas.

Se espera que este trabajo sea beneficioso para el campo industrial y artesanal de la pesca y los científicos marinos, así como un punto de partida útil donde la motivación principal de los autores es generar la concientización respecto a la identificación de los principales depredadores relacionado al rescate y liberación de las especies marinas que interactúan con la pesca objetivo como es la anchoveta, jurel y caballa.[Link al White Paper](https://drive.google.com/file/d/1WY2wJVpQvEA8-5xLBjGibL4HIeXe-E-A/view?usp=share_link)
